# CLAUDE.md — TaxReady Canadian Tax Prep Guide
# Read this file at the start of every Claude Code session.

## Project Overview
TaxReady is a KYC-style interview tool that generates a personalized T1 preparation
brief for Ontario residents filing their 2025 Canadian tax return. Output is a
checklist only — no tax computation.

## Stack
- Next.js 14, TypeScript, Tailwind CSS
- Railway deployment (nixpacks, npm run start)
- OpenRouter AI (server-side only, model via AI_MODEL env var)
- No database — static JSON files only

## How to Run
```bash
npm run dev          # development server on localhost:3000
npm run build        # production build
npm run start        # production server
npx tsx src/lib/validateTaxonomy.ts   # validate taxonomy JSON
node --import tsx --test              # run unit tests
```
IMPORTANT: Always use `tsx` runner, never `ts-node`. ESM config requires tsx.

## Critical Rules
1. NEVER hand-edit interview-flow.json — it is a build artifact
2. Each branch lives in its own file under taxonomy/2025/branches/
3. Always run validateTaxonomy.ts after changing any JSON
4. Always run the test suite after changing session.ts or matcher.ts
5. tsx runner only — never ts-node
6. OpenRouter API key never exposed to client — server-side only via /api/clarify

## Common Tasks
- Add/edit a branch: edit the file in taxonomy/2025/branches/, then run buildInterviewFlow.ts
- Add a taxonomy node: edit taxonomy/2025/items.json, run validateTaxonomy.ts
- Fix a screen: edit the relevant branch file, rebuild, test
- Run the app end-to-end: npm run dev, open localhost:3000
